# VULKAN GAME ENGINE
![](showcase_gif.gif)



## Description
A game engine programmed in C using Vulkan as a didactic exercise to learn game engine development at its lowest level.

## Features
+ Multithread
+ Jobs system
+ Memory system
+ Hashmap
+ Dynamic array
+ Freelist
+ Custom allocators
+ Vulkan renderer
+ Render-view system
+ Material system
+ Shader system
+ Geometry system
+ Resource system
+ Texture system
+ Camera system
+ Inputs
+ Events
+ Logger
+ String
+ Math library
+ Platform arquitecture
+ File system
+ DLL arquitecture library
+ Skybox
+ Directional light and point light
+ Normal map, diffuse map, specular map
+ OIT renderer
+ Toolchain 
  
## Installation
You must install LunarG Vulkan SDK, and after that:
+ Open the root directory with Visual Studio Code
+ Press `CTRL + SHIFT + B` to compile
+ Finally press `F5`.

If you don't use Visual Studio Code, execute `build-all.bat`
